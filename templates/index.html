Welcome to the Stair Calculator

Our stair calculator helps you design and calculate the dimensions of your stairs easily and accurately.

How to Use the Calculate-Stairs API

  1. Send a POST request to the endpoint with a valid API Key on /calculate-stairs with the following JSON payload: 
    {
    "height": 20,
    "depth": 25,
    "margin": 3, (optional)
    "maxStepDepth": 35, (optional)
    "minStepDepth": 25 (optional)
    "maxStepHeight": 20, (optional)
    "minStepHeight": 10, (optional)
}
  2. The API will respond with a JSON object containing the calculated dimensions of the stairs, such as: 
    {
    "stepDepth": 12.5,
    "stepHeight": 7.0,
    "stepNum": 2
}
  3. Use the calculated dimensions to build your stairs.
